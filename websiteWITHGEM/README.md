# Sortacle Sustainability Dashboard (with Gemini)

This is a beautifully stylized frontend for the Sortacle project, focused on sustainability and data visualization.

## Features

- **Sustainability Theme**: Modern, clean UI with eco-friendly aesthetics.
- **Interactive Tabs**:
    - **Overview**: General project status and impact.
    - **Heatmaps**: Spatial data visualization for sorting efficiency.
    - **Dashboards**: Real-time metrics and analytics.
    - **Sponsor UI**: Dedicated portal for project partners and sponsors.
- **Responsive Design**: Works on mobile, tablet, and desktop.

## Technologies

- HTML5 / CSS3 (Modern Flexbox/Grid)
- JavaScript (Tab management & UI interactions)
- Lucide Icons (CDN)
- Google Fonts (Inter & Playfair Display)

## Setup

Simply open `index.html` in any modern web browser to view the dashboard.

```bash
# To serve locally
python3 -m http.server 8080
```
