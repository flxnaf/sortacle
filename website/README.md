## üõ†Ô∏è Localhost Preview

You can easily preview this dashboard locally using any of the following methods:

### Method 1: Python (Recommended)
If you have Python installed, run this command inside the `website` folder:
```bash
python3 -m http.server 8080
```
Then visit `http://localhost:8080` in your browser.

### Method 2: Node.js (serve)
If you have Node.js and `serve` installed:
```bash
npx serve .
```
Then visit `http://localhost:3000`.

### Method 3: Visual Studio Code (Live Server)
If you are using VS Code, install the "Live Server" extension and click the **Go Live** button in the bottom right corner of the status bar.
